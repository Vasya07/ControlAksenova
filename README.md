# ControlAksenova — Расчёт оптимального веса

[![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=csharp&logoColor=white)](https://learn.microsoft.com/en-us/dotnet/csharp/)
[![WPF](https://img.shields.io/badge/WPF-0C54C2?style=for-the-badge&logo=windows&logoColor=white)](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/)
[![.NET Framework](https://img.shields.io/badge/.NET_Framework-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)](https://dotnet.microsoft.com/en-us/download/dotnet-framework)

**WPF-приложение** для вычисления оптимального веса человека на основе пола, роста и веса. Программа включает проверку ввода, расчёт отклонения от нормы и цветовую оценку результата (±3 кг).

## Функциональные возможности

- Выбор пола (Мужчина / Женщина);
- Расчёт нормального веса по формулам:
  - **Мужчины:** `(рост - 100) * 1.13`;
  - **Женщины:** `(рост - 100) * 0.90`;
- Проверка допустимых значений:
  - Рост: **130–220 см**;
  - Вес: **40–170 кг**;
- Оценка веса с учётом **±3 кг** от нормы:
  - «Ниже нормы» (отклонение < -3 кг);
  - «Норма» (отклонение в пределах ±3 кг);
  - «Выше нормы» (отклонение > +3 кг);
- Цветовая индикация результата (зелёный, оранжево-красный, тёмно-оранжевый);
- Обработка ошибок ввода (нечисловые значения, выход за пределы диапазонов).

## Запуск проекта

### Требования
- **Visual Studio 2026** (с нагрузкой `.NET desktop development`)
- **.NET Framework 4.7.2** или выше

### Инструкция по запуску
1. Клонируйте репозиторий:
```
git clone https://github.com/Vasya07/ControlAksenova.git
```
2. Откройте решение **ControlAksenova.slnx** в Visual Studio;
3. Соберите проект: Build → Build solution;
4. Запустите приложение: Debug → Start without debugging.

## Тестирование
Проект содержит unit-тесты (MS Test), которые покрывают:

* Логику расчёта нормального веса для мужчин и женщин;
* Определение оценки веса на основе отклонения;
* Формирование строки отклонения (включая нулевое отклонение);
* Граничные значения роста и веса;
* Валидацию числового ввода;
* Комбинированные сценарии.

## Запуск тестов
1. Откройте Test Explorer: Test → Test Explorer;
2. Нажмите Run All Tests;
3. Все тесты должны быть с зелёной галочкой.

## Пример использования
1. Входные данные: Рост: 180 см, Вес: 80 кг, Пол: Мужчина. Результат: Норма: 90,4 кг, Отклонение: -10,4 кг, Оценка: ниже нормы (оранжевый);
2. Входные данные: Рост: 165 см, Вес: 60 кг, Пол: Женщина. Результат: Норма: 58,5 кг, Отклонение: +1,5 кг, Оценка: норма (зелёный);
3. Входные данные: Рост: 160 см, Вес: 54 кг, Пол: Женщина. Результат:	Норма: 54,0 кг, Отклонение: отсутствует, Оценка: норма.

## Обработка ошибок
При вводе некорректных данных программа показывает информативные сообщения:
* Нечисловой ввод: "Пожалуйста, введите рост/вес в виде числа (десятичные дроби через запятую)";
* Выход за диапазон роста: "Выход роста за пределы допустимого диапазона! Разрешённый диапазон: 130 см - 220 см";
* Выход за диапазон веса: "Выход веса за пределы допустимого диапазона! Разрешённый диапазон: 40 кг - 170 кг".
